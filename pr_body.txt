### Description
This PR establishes a comprehensive internal project management and repository intelligence system for the Two Second Witness project. It transforms the repository into an 'external memory' for both the human project owner and future AI coding sessions.

### What was created:
1.  **PROJECT_COMMAND_CENTER.md**: A root-level human-readable dashboard showing current status, roadmap progress, and next actions.
2.  **.github/project-state.yml**: A machine-readable file for AI agents to track project state.
3.  **docs/product/development-continuity/AUTOMATED_PROJECT_AUDIT.md**: A human-friendly audit report (baseline created; automated workflow template provided).
4.  **docs/product/development-continuity/START_NEXT_SESSION.md**: A tool to generate the recommended prompt for bootstrapping future AI sessions.
5.  **docs/product/development-continuity/HUMAN_PROJECT_WORKFLOW.md**: A guide for the project owner on how to use these new tools.
6.  **Updates to existing continuity docs**: Integrated the new systems into the Context Bootstrap and Implementation State documents.

*Note: The GitHub workflow file (.github/workflows/project-audit.yml) could not be automatically created due to GitHub App permission restrictions. A template has been provided in the documentation for manual installation.*

### Why it exists:
To allow for seamless development continuity. As the project evolves through its 10-update roadmap, this system ensures that no context is lost between sessions and that both humans and AI agents always know 'Where are we?', 'What is next?', and 'What decisions are protected?'.

### How humans use it:
1. Open `PROJECT_COMMAND_CENTER.md` to see current status.
2. Use `START_NEXT_SESSION.md` to kick off an AI agent.
3. Review PRs against the roadmap and locked decisions.

### How future AI sessions use it:
AI agents are instructed by the bootstrap prompt to read the Command Center and State file first. This provides them with immediate context on the current milestone and prevents them from making unrelated or unauthorized changes.
