# Challenge Type Specifications

Every production Challenge Type requires the [`CHALLENGE_TYPE_ACCEPTANCE_CONTRACT.md`](CHALLENGE_TYPE_ACCEPTANCE_CONTRACT.md), the detailed [`CHALLENGE_TYPE_SPEC_TEMPLATE.md`](CHALLENGE_TYPE_SPEC_TEMPLATE.md), and a distinct position in the [`CHALLENGE_TYPE_PORTFOLIO_MATRIX.md`](CHALLENGE_TYPE_PORTFOLIO_MATRIX.md).

## Production Challenge Types

- [`SCENE_INVESTIGATION_SPEC.md`](SCENE_INVESTIGATION_SPEC.md) — Office, Kitchen, Workshop
- [`FLASH_WORDS_SPEC.md`](FLASH_WORDS_SPEC.md) — Single Word, Pair Order, Word Stream
- [`SPOT_THE_DIFFERENCE_SPEC.md`](SPOT_THE_DIFFERENCE_SPEC.md) — Presence, Attribute, Sequential, Arrangement
- [`OBJECT_RECALL_SPEC.md`](OBJECT_RECALL_SPEC.md) — Seen Set, Missing Set, Position Group
- [`PATTERN_RECALL_SPEC.md`](PATTERN_RECALL_SPEC.md) — Grid Path, Shape Sequence, Pattern Build

Style guides and visual reviews remain beside their family specifications. Phase 5 visual artifacts are under `docs/product/artifacts/phase5_challenge_types/`.
